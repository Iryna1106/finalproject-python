# Personal Assistent (CLI)

Консольний-бот помічник для керування контактами та нотатками

## Основні можливості
Застосунок складається з двох основних модулів:
1. **Адресна книга (Address Book):** збереженння імен, номерів телефонів, електронної пошти, фізичних адрес та днів народження.
2. **Нотатник (Notes):** збереження текстових нотаток із можливістю додавання тегів, пошуку та сортування.

**Особливості**
- Інтерективний інтерфейс командного рядка (CLI) з автодоповненням команд (через клавішу `Tab`).\
- Валідація введених даних (формат телефону, email, дати народження).
- Надійне збереження даних на жорсткий диск.

---

## Встановлення 

Для роботи програми потрібен **Python 3.8** або вище.

1. Склонуйте репозиторій на свій комп'ютер:

**git clone <посилання_на_репозиторій>**

2. Перейдіть у папку проекту:

**cd finalproject-python**

3. Встановіть необхідні залежності:

**pip install -r requirements.txt**

## Використання 

Щоб запустити помічника, виконайте команду в терміналі: 

**python main.py**

## Доступні команди

    Контакти (Адресна книга)
add <name> <phone> — Додати новий контакт (або телефон до існуючого).

change <name> <old_phone> <new_phone> — Змінити номер телефону.

phone <name> — Показати телефони контакту.

all — Показати всі контакти.

search <query> — Знайти контакт за ім'ям/телефоном/email.

delete <name> — Видалити контакт.

add-birthday <name> <DD.MM.YYYY> — Додати день народження.

show-birthday <name> — Показати день народження контакту.

birthdays [days] — Показати дні народження на найближчі days днів (за замовчуванням 7).

add-email <name> <email> — Додати електронну пошту.

## Нотатки

add-note — Створити нотатку.

all-notes — Показати всі нотатки.

find-note <query> — Знайти нотатки за текстом.

edit-note <id> — Редагувати нотатку.

delete-note <id> — Видалити нотатку.

add-tag <id> <tag> — Додати тег.

find-tag <tag> — Знайти за тегом.

sort-notes-by-tags — Сортувати за тегами.

## Загальні команди

help — Показати список команд.

exit або close — Зберегти дані та вийти.
